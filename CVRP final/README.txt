ALGORITMO  COMPLETO SE BASEIA NESTES PASSOS:
  
    1. PROCURAR DESTINO
        - PROCURA CASA MAIS PRÓXIMA
        - VER SE É COMPATÍVEL COM A CAPACIDADE RESTANTE DO CARRO
            - SE FOR FAZ A ENTREGA
            - SE NÃO FOR PROCURA A PRÓXIMA CASA
            - SE NO FIM NÃO HOUVER MAIS CASA COMPATÍVEL VOLTA PARA A 			EMPRESA SOMANDO A DISTÂNCIA
    2. FAZER ENTREGA
        - DIMINUIR CAPACIDADE DO CARRO
        - COLOCA A DEMANDA DA CASA PARA ZERO
        - SOMAR DISTÂNCIA PERCORRIDA
    3. REPETIR
